# Services Layer

This folder contains business logic and external service integrations for the FastAPI backend.

## Services

### Core Services
- **feedback_service.py** - User feedback processing and storage
- **quote_service.py** - Quote management operations
- **post_quote_service.py** - Quote creation and validation

### AI Integration Services
- **Gemini/** - Google Gemini AI integration
- **Langraph_reactflow/** - LangGraph workflow integration
- **LanggrapProject/** - LangGraph project implementations

## Features
- Clean separation of business logic
- External API integrations
- Async service patterns
- Error handling and validation
- Service abstraction layer

## Purpose
Centralized business logic and service integrations providing clean interfaces between API routes and external systems.
