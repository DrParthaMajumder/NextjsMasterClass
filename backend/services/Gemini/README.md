# Gemini AI Services

This folder contains Google Gemini AI integration services.

## Services

### interact_with_gemini.py
- Core Gemini AI integration using LangChain
- Structured JSON response generation
- Prompt engineering and response formatting
- Async AI processing capabilities

### interact_with_gemini_streaming.py
- Streaming AI responses implementation
- Real-time chat functionality
- Server-sent events for live responses
- Optimized for conversational AI

## Features
- LangChain integration with Google Gemini
- Structured JSON output parsing
- Async/await patterns
- Error handling and validation
- Streaming response support
- Prompt template management

## Configuration
Requires `GEMINI_API_KEY` environment variable for authentication.

## Purpose
Comprehensive Google Gemini AI integration providing both standard and streaming AI capabilities for content generation, chat, and intelligent responses.
