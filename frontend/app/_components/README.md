# Components

This folder contains shared React components used throughout the application.

## Components

### Header.tsx
- Main navigation header component
- Includes navigation links and branding

### Footer.tsx
- Footer component with copyright and links
- Consistent footer across all pages

## Usage
These components are imported and used in layout.tsx and various pages throughout the application.

## Purpose
Centralized reusable components for consistent UI/UX across the Next.js application.
