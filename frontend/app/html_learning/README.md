# HTML Learning Examples

This folder contains HTML fundamentals and form examples with Tailwind CSS styling.

## Lessons

### Ex1-Ex5 - Basic HTML Elements
- Headings, paragraphs, lists
- Links and images
- Tables and basic structure

### Ex6-Ex10 - Forms Fundamentals
- Input types and form elements
- Form validation and styling
- Focus states and accessibility

### Ex11-Ex15 - Advanced Forms
- Complex form layouts
- Form validation patterns
- Responsive form design

### Ex16-Ex20 - HTML5 Features
- Semantic HTML5 elements
- Media elements (audio/video)
- Embedded content and iframes
- Modern HTML features

## Features
- Progressive complexity
- Tailwind CSS integration
- Accessibility considerations
- Responsive design examples

## Purpose
Learn HTML fundamentals from basic elements to advanced HTML5 features with modern styling practices.
