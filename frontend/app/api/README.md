# API Routes

This folder contains Next.js API routes for backend functionality.

## Routes

### askGemini/
- AI chat integration with Google Gemini
- POST endpoint for AI responses

### feedback/
- User feedback collection
- POST endpoint for feedback submission

### flowchart/
- Flowchart and diagram generation
- Test flowchart examples

## Features
- Serverless API endpoints
- AI integration capabilities
- Data collection and processing
- External service integrations

## Purpose
Backend API endpoints supporting frontend functionality including AI services, user feedback, and data processing.
