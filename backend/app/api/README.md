# API Routes

This folder contains API route definitions and handlers.

## Structure

### routes/
- Version-specific route definitions (v1)
- Individual router modules for different features
- Request/response handling logic

## Features
- Modular route organization
- Version control for API endpoints
- Clean separation of concerns
- Easy maintenance and scaling

## Purpose
Organized API route structure following FastAPI patterns for maintainable and scalable API development.
