# Custom Hooks

This folder contains custom React hooks for reusable stateful logic.

## Hooks

### useAskGemini.ts
- Custom hook for Google Gemini AI integration
- Handles loading states, errors, and responses
- Provides clean interface for AI API calls

## Features
- Async state management
- Error handling patterns
- Loading state management
- API integration abstraction

## Purpose
Reusable logic extracted into custom hooks following React best practices for component composition.
